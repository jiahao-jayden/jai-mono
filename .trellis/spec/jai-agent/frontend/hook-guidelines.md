# Hook Guidelines

> Frontend hook patterns for `@jayden/jai-agent`.

---

## Not Applicable

`@jayden/jai-agent` is a backend-only package. It contains no React hooks.

Note: The `HookRegistry` class in this package is a backend concept (beforeToolCall/afterToolCall pipeline), not a React hook. See `@jayden/jai-desktop` for frontend hook guidelines.
