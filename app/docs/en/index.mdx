# Introduction

This site is the starting point for Jai documentation.

Today it contains a single introduction page in English and Chinese. As the project grows, this site can expand into onboarding guides, reference material, and examples without changing the core structure.

## What you can expect here

- A clear overview of the product and workspace
- Guides for common tasks and setup flows
- Reference content for stable concepts and interfaces
- Examples that show real usage patterns

## Current status

The documentation structure is ready for multilingual content, with English as the default language and Chinese available from the language switcher.
